# Air-Quality-Forecasting-for-Predictive-Analysis-Using-Machine-Learning-
This study forecasts Chennai’s Air Quality Index (AQI) using machine learning models—Random Forest, XGBoost, LightGBM, and SVR. By analyzing pollutants such as PM2.5, PM10, NO₂, CO, O₃, and SO₂, it provides reliable predictions to support authorities and citizens.
